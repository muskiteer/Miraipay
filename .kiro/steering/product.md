# Product Overview

Miraipay is an AI-powered marketplace that enables developers to monetize APIs through autonomous AI agent interactions. Users interact with tools via natural language, and payments are processed automatically using the X402 protocol and MNEE stablecoin (ERC-20 token on Ethereum).

## Core Value Proposition

- Developers register APIs as monetizable tools with custom pricing
- Users chat with an AI agent (powered by Google Gemini) to discover and use tools naturally
- AI agent automatically selects appropriate tools, processes payments, and executes API calls
- All transactions are tracked transparently with MNEE token payments

## Key Features

- AI-first tool discovery and execution via natural language chat
- X402 payment protocol for seamless micropayments
- Automatic Ethereum wallet generation with AES-256 encrypted private keys
- Admin approval workflow for tool quality control
- Real-time transaction history and earnings tracking
- Demo tools included: Movie Ticket Booking, QR Code Generator

## User Roles

- **Developers**: Create and monetize API tools, track earnings
- **Users**: Use tools via AI agent chat, manage MNEE balance
- **Admins**: Approve/reject tools, manage users and platform health
